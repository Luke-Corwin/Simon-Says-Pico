# Simon Says for Raspberry Pi Pico

A super beginner-friendly Simon Says memory game built with MicroPython and a Raspberry Pi Pico.

This project was created to learn the basics of embedded systems programming, GPIO pins, buttons, LEDs, buzzers, and game logic in a fun and approachable way.

## Features

* Random Simon Says sequences
* 4 colored LEDs
* 4 push buttons
* Buzzer sound effects
* Startup animation
* Score tracking in the console
* Automatic startup when saved as `main.py`
* Beginner-friendly code with easy-to-understand functions

## Hardware Used

* Raspberry Pi Pico
* Breadboard
* 4 LEDs
* 4 push buttons
* Active buzzer
* Resistors
* Jumper wires

## Files

* `main.py` - Main Simon Says game code
* `config.py` - GPIO pin configuration
* `README.md` - Project documentation
* `images/` - Photos and wiring images

## How to Run

1. Flash MicroPython onto the Raspberry Pi Pico.
2. Open the project in Thonny.
3. Save `main.py` to the Pico.
4. Run the program or save it as `main.py` on the Pico for automatic startup.
5. Repeat Simon's sequence and see how high you can score!
